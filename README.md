# 碧蓝档案主题西蒙游戏 (Blue Archive Simon Game)
一款基于经典西蒙游戏（Simon Game）改编的网页版小游戏，融入碧蓝档案（Blue Archive）角色元素，包含角色语音、关卡进度、自定义设置等趣味功能。

## 项目介绍
西蒙游戏是一款经典的记忆序列游戏，本项目以碧蓝档案为主题，替换了传统的颜色按钮为碧蓝档案角色形象，并增加了角色语音、BGM、自定义游戏参数等功能，通过网页端即可游玩，适配移动端与桌面端。

### 核心特色
- 🎨 碧蓝档案角色主题：使用 Momoi、Arisu、Iroha、Izuna 角色形象替代传统颜色按钮
- 🔊 角色语音+音效：点击对应角色按钮播放专属语音，操作反馈音效（正确/错误）
- 🎵 可自定义BGM与角色音量
- ⚙️ 灵活的游戏设置：支持调整序列演示延时、过渡动画效果
- 📱 响应式布局：适配手机、平板、桌面等不同设备
- 📈 关卡进度展示：实时显示当前关卡与进度条

## 技术栈
- 前端基础：HTML5、CSS3、JavaScript (ES6+)
- 样式框架：Bootstrap 5.3.3
- 音频处理：原生 Web Audio API
- 布局：Flexbox、CSS Grid、响应式设计

## 快速开始
### 运行方式
1. 克隆本仓库到本地
   ```bash
   git clone <仓库地址>
   cd ba-simon-game
   ```
2. 直接打开 `index.html` 文件（推荐使用本地服务器，避免音频跨域问题）
   - 简易本地服务器（Node.js）：
     ```bash
     # 安装 http-server（若未安装）
     npm install -g http-server
     # 启动服务器
     http-server ./ -p 8080
     ```
3. 打开浏览器访问 `http://localhost:8080` 即可开始游戏

### 游戏规则
1. 点击「主菜单」-「开始吧！」启动游戏
2. 每一关会先演示一组随机的角色序列（伴随语音/动画）
3. 按照演示的顺序点击对应角色按钮，完全正确则进入下一关
4. 若点击顺序错误，游戏结束并展示结果（当前关卡、正确序列、你的操作序列）
5. 可通过主菜单调整音量、序列延时、过渡效果等设置

## 功能说明
### 界面元素
- **关卡进度**：顶部显示当前关卡，进度条展示关卡完成状态
- **角色按钮**：四个角色（黄/蓝/红/绿）对应不同语音与交互
- **主菜单**：包含音量调节、序列延时、过渡效果设置、全屏模式
- **结果弹窗**：游戏结束后展示关卡、正确序列、玩家操作序列，支持重玩/返回主菜单

### 自定义设置
| 设置项         | 说明                     | 取值范围          | 默认值                          |
|----------------|--------------------------|-------------------|---------------------------------|
| 角色音量       | 调整角色语音播放音量     | 0.0 ~ 1.0         | 0.3                             |
| BGM音量        | 调整背景音乐音量         | 0.0 ~ 1.0         | 0.6                             |
| 顺序延时       | 演示序列时的按钮间隔时间 | 100ms ~ 1000ms    | 600ms                           |
| 过渡效果       | 按钮动画的过渡曲线       | 多种预设/自定义贝塞尔曲线 | cubic-bezier(0.68, -0.55, 0.27, 1.55) |
